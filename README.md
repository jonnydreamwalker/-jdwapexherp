## Technical SEO & Performance

- JSON-LD Organization + LocalBusiness on homepage
- JSON-LD Product + Offer on Hardscape, Lighting, Substrates
- Open Graph + Twitter cards
- `theme-color: #000000` (matte black browser chrome)
- Favicon + apple-touch-icon
- `pragmatic-rendering`, googlebot/bingbot max-snippet flags
- Prefetch / preconnect for video and payment nodes
- `content-visibility: auto` on product cards
- Selection highlight uses brand emerald variables
